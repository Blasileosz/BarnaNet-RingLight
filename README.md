# BarnaNet - RingLight

## Overview
- ESP device controlling [neopixel rings](https://www.emag.hu/rgb-led-gyuru-16-bites-ws2812-5050-ai1207/pd/D9MCXVMBM/)
- Designed to be used in parallel wit other types of BarnaNet devices
